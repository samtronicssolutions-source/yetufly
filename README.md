# Yetu E-commerce

A fully functional e-commerce website with M-Pesa integration.

## Features
- Product browsing and search
- Shopping cart with local storage
- M-Pesa STK Push payments
- Admin dashboard with product management
- Responsive design

## Tech Stack
- Node.js + Express
- MongoDB Atlas
- Vanilla JavaScript
- M-Pesa API (Safaricom)

## Installation

```bash
npm install
cp .env.example .env
# Update .env with your MongoDB URI
npm run seed
npm start
