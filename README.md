# Yetu E-commerce

A fully functional e-commerce website with M-Pesa integration.

## Features
- Product browsing and search
- Shopping cart
- M-Pesa STK Push payments
- Admin dashboard
- Responsive design

## Tech Stack
- Node.js + Express
- MongoDB Atlas
- Vanilla JavaScript

## Installation
```bash
npm install
cp .env.example .env
# Update .env with your MongoDB URI
npm run seed
npm run add-products
npm start
