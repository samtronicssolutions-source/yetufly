# Yetu E-commerce

A fully functional e-commerce website with M-Pesa integration, built with Node.js, Express, MongoDB, and vanilla JavaScript.

## Features

- 🛍️ Product browsing and search
- 🛒 Shopping cart functionality
- 💳 M-Pesa payment integration (STK Push)
- 👑 Admin dashboard
- 📱 Responsive design
- 🔍 SEO friendly

## Tech Stack

- **Backend**: Node.js, Express.js
- **Database**: MongoDB Atlas
- **Frontend**: HTML5, CSS3, JavaScript
- **Payments**: M-Pesa API (Safaricom)
- **Deployment**: Render.com
